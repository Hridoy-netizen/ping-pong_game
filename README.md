# ping-pong_game

A classic, interactive Ping Pong Game built to deliver a smooth and engaging user experience. This project brings the timeless arcade experience straight to your screen with responsive controls and sleek visuals.

🚀 Live Demo
👉 View the Live Project (Be sure to enable GitHub Pages in your repo settings to activate this link!)

✨ Features
Smooth Gameplay: Optimized frame rates for fluid paddle and ball movements.

Responsive Controls: Quick and intuitive inputs for an authentic arcade feel.

Score Tracking: Real-time score updates to keep track of the winner.

Clean UI: A minimalist, modern aesthetic that keeps the focus entirely on the game.

🛠️ Tech Stack
HTML5: For the core structure and game canvas.

CSS3: For modern styling and layout layout design.

JavaScript (ES6): For the game logic, physics engine, and collision detection.

🎮 How to Play
Launch the Game: Open the live link or run the project locally.

Controls:

Player 1 (Left): Use the W (Up) and S (Down) keys.

Player 2 / AI (Right): Use the Up Arrow and Down Arrow keys.

Objective: Defend your side! Prevent the ball from passing your paddle while trying to score against your opponent. The first player to reach the score limit wins.
